# 快速开始指南

## 第一步：初始化配置

首次使用时，需要设置学期开始日期。

### 方法 1：自然语言对话

直接告诉我：
```
我的这学期是 3 月 1 号开始的
```

我会自动保存配置。

### 方法 2：手动设置

编辑 `/root/.openclaw/workspace/calendar/settings.json`：
```json
{
  "semesterStart": "2026-03-01",
  "timezone": "Asia/Shanghai"
}
```

---

## 第二步：上传课表

直接上传课表图片（截图或照片），我会自动识别并添加。

**支持格式**：
- 本科生课表（包含教师姓名）
- 研究生课表（课程名 + 地点）

**识别准确率**：90%+（USTC 专属优化）

---

## 第三步：开始使用

### 查询日程

```
今天有什么课？
明天呢？
这周安排
下周三下午有空吗？
```

### 添加事件

```
明晚 7 点去图书馆
后天下午 3 点和朋友出去玩
提醒我下周一交作业
```

### 修改/删除

```
把明天的图书馆改成 8 点
取消周五的计划
```

---

## 第四步：设置提醒（可选）

提醒默认开启，每 30 分钟检查一次。

如需自定义，创建 cron 任务：

```bash
# 每 30 分钟检查
openclaw cron add --schedule '*/30 * * * *' \
  --payload '{"kind": "agentTurn", "message": "检查日历提醒"}'
```

## 常见问题

### Q: 识别错误怎么办？
A: 重新上传清晰的课表截图，或手动告诉我课程信息。

### Q: 如何修改学期开始日期？
A: 直接说"我的学期是 2 月 25 号开始的"，我会更新配置。

### Q: 提醒太频繁/不频繁？
A: 告诉我"提前 1 小时提醒"或"提前 10 分钟提醒"，我会调整。

### Q: 支持 QQ/微信推送吗？
A: 当前仅支持当前聊天，后续会添加多渠道推送。
